# StevTech Tools

Free desktop utilities from [StevTech](https://stevtech.com.au) for customers
working with our services. Each tool lives in its own folder under `tools/`,
and ready-to-run Windows downloads are published on the
[Releases page](../../releases).

## Tools

| Tool | What it does | Download |
|---|---|---|
| [Spray Packager](tools/spray-packager/) | Prepares DJI Terra spray-drone output (`Result.tif` + `Segment.tif`) for submission to StevTech: compresses the orthomosaic for fast viewing and bundles everything into a single zip. | `SprayPackager.exe` from the [latest release](../../releases/latest) |

## Getting a tool

1. Open the [Releases page](../../releases/latest) and download the `.exe`
   for the tool you need. Nothing else to install.
2. Double-click it and follow the tool's instructions (each tool's folder has
   a README with step-by-step usage).

**"Windows protected your PC" warning:** the tools are not yet code-signed, so
Windows SmartScreen may show a blue warning the first time you run one. Click
**More info → Run anyway**. The download is safe as long as it came from this
page.
